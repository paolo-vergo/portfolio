Project 1: Conformal Prediction Packages

April 28, 2022

Conformal Prediction for regression is a powerful technique that allows us to construct prediction sets with a high level of reliability. These prediction sets come with a marginal coverage guarantee at a specified level, typically denoted as 1-α. This means that we can have a certain degree of confidence in the predictions made by the model.

During my thesis, I delved into the application of Conformal Prediction for regression in two distinct scenarios: multivariate responses and functional responses. In the multivariate case, I explored situations where the outcome variable is composed of multiple correlated variables. This can arise, for example, in finance when predicting stock prices based on various market indicators or in medical research when predicting multiple health outcomes based on patient characteristics.

In the functional case, I focused on scenarios where the outcome variable is represented by a function rather than a single numeric value. This is common in fields such as signal processing, where we aim to predict time series or other continuous signals. By extending Conformal Prediction to handle functional responses, we can make reliable predictions for such complex and dynamic data.

To facilitate the practical implementation and usage of Conformal Prediction in these two contexts, I developed dedicated R packages. The first package, called "conformalInference.multi," is tailored for handling multivariate responses. It provides a user-friendly interface and a set of functions specifically designed for this type of data.

Similarly, the second package, "conformalInference.fd," is dedicated to the analysis of functional responses. It includes a comprehensive set of tools and algorithms that enable users to construct prediction sets and make reliable predictions for functional data.

I'm pleased to announce that both packages have been submitted and are now publicly available on CRAN, a central repository for R packages. This means that researchers and practitioners interested in leveraging Conformal Prediction for regression in the multivariate or functional context can readily access and utilize these packages in their own work.

For those interested in exploring the inner workings and implementation details, I have provided the complete code for both packages on GitHub. The repositories, "conformal.multi" and "conformal.fd," contain the source code, documentation, and examples to guide users in applying the packages effectively.

By providing these packages and resources, I aim to empower researchers and practitioners to make reliable predictions in these challenging data scenarios and advance the field of Conformal Prediction in regression analysis.
